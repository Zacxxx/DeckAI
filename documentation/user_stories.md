# User Stories

## Creating with Magic (Generation)
- **As a user**, I want to drop in my messy meeting notes and a link to my company's website, so the AI can ingest the brand guidelines and structure a beautiful, fully branded 15-slide 16:9 presentation completely automatically.
- **As a user**, I want the AI to handle all pagination logically. If my text is too long for one A4 page, the AI intelligently splits the content, maintaining the layout rhythm and typography across multiple pages.
- **As a power user (Local Sovereignty)**, I want to toggle the system into "Local Mode", running the opencode harness locally and pointing it at OpenRouter. This allows me to use any LLM API I choose and keeps my proprietary documents entirely off managed cloud services.

## Granular Steering & The Designer Experience
- **As a user**, I want to click an image container, type "make it punchier with a glassmorphism feel", and watch the AI precisely rewrite the CSS filters and borders for that single element without disturbing the rest of the slide's harmony.
- **As a developer/advanced user**, I want to switch the view to let the AI output pure React code for a particular generative component so I can integrate it back to my own SaaS web app, bridging document generation and software design.

## Modularity & Extraction
- **As a product owner**, I want to select a composite element (e.g., a complex "Pricing Table" layout) and click "Save to Design System". The AI extracts the underlying structural logic and tokens, so my team can deploy it in future proposals by just saying "use our standard pricing table".
- **As a user**, I expect the AI to automatically identify when a background image is conflicting with text readability and autonomously apply a subtle gradient overlay or adjust typography color without me asking.

## Zero Friction Exports
- **As a user**, I want to export my generative slide deck to an editable PPTX, and I expect the AI-generated charts to be native PowerPoint charts—not just flattened images—so I can tweak the underlying data manually if needed.
- **As a user**, I want my PDF export to reflect the exact pixel alignments I saw in the browser canvas—a 100% WYSIWYG mathematical guarantee.
